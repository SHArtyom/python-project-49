# Description

This project is a small entartainment package of 5 console mathematical games:

- brain-even
- brain-calc
- brain-gcd
- brain-progression
- brain-prime
____

## Minimum requirements

`python3.9`
____

## Installation guide

To install poetry execute

`make install` 

To setup the package execute

`make package-install`
____
